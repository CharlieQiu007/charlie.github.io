使用 Hugo 和 GitHub 搭建博客

Hugo

1. 下载 hugo

https://github.com/gohugoio/hugo/releases

2. 安装

a. 解压安装包,放在指定的位置


b. 设置环境变量


c. 生成本地网站

$ hugo new site test.com

会有如下提示

Congratulations! Your new Hugo site is created in C:\Users\[用户名]\test.com.

进入 C:\Users\[用户名]\test.com

主题

1. 下载

https://www.gohugo.org/theme/

2. 使用

a. 将主题文件夹重新命名，把主题文档 hugo-strata-theme-master 命名为 hugo-strata-theme；
b. 将 hugo-strata-theme 全部复制到 C:\Users\[用户名]\test.com\themes 目录下
c. 将下载好的主题解压，把其中 exampleSite 中的文档全部复制到 C:\Users\[用户名]\test.com 下，替换 C:\Users\[用户名]\test.com 下的文件

本地运行

1. 打开命令行

进入 C:\Users\[用户名]\test.com

$ hugo server

提示

Web Server is available at http://localhost:10946/ (bind address 127.0.0.1)

浏览器访问 http://localhost:10946/ 发现本地运行成功了

部署

1. Github Pages

部署在 GitHub Pages 上，首先在GitHub上创建一个Repository，命名为：coderzh.github.io （coderzh替换为你的github用户名）。

2. 生成 public

$ hugo --baseUrl="http://coderzh.github.io/"

3. 上传网站

$ cd public
$ git init
$ git remote add origin https://github.com/coderzh/coderzh.github.io.git
$ git add -A
$ git commit -m "first commit"
$ git push -u origin master
